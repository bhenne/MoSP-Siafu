Mobile Security & Privacy extended Siafu Simulator

The Mobile Security & Privacy (MoSP) extended Siafu Simulator is an extention of the Siafu Open Source Context Simulator for the research in the field of security and privacy of modern mobile devices. It has been developed for the simulation of indoor sub-scenarios of simulations using the Mobile Security & Privacy Simulator. The development of the simulator originated at the Distributed Computing & Security Group at the Leibniz Universit√§t Hannover, Germany.

Some extended features:

  • simulation of the inner of buildings with multiple stories
  • near-by detection of other agents across multiple stories
  • interface for the external control (stepping, adding and removing agents) of a simulation
    • tailored for the use as coupled MoSP simulation
    • see controller implementation

Examples

A demo video of an exemplary coupled simulation with a MoSP road-level simulation and three indoor sub-simulations can be found at YouTube.
